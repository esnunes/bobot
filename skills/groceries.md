---
name: groceries
description: Manage grocery shopping lists
---
When the user wants to manage their grocery list, use the task tool with project "groceries".

Examples:
- "Add milk to groceries" → create task "milk" in project "groceries"
- "What's on my grocery list?" → list tasks in project "groceries"
- "I got the eggs" → update task "eggs" to status "done"
- "Remove bread from the list" → delete task "bread"
